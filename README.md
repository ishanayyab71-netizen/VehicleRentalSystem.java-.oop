# 🚗 Vehicle Rental System - Java OOP Project

## 📋 Project Overview
A premium desktop GUI application for managing vehicle rentals, built using Java Swing and core OOP principles. Features secure login, inventory management, and automated billing.

## ✨ Key Features
- **🔐 Secure Login System** - Username: `Isha`, Password: `Isha123`
- **🚘 Vehicle Management** - Add, Update, Delete, Search Cars & Bikes
- **📊 Live Dashboard** - Real-time count of total vehicles, cars, bikes
- **🧾 Auto Invoice Generation** - Calculates rent and generates formatted bills
- **💾 File Handling** - Saves all data to `data.txt` for persistence
- **🎨 Premium UI** - Modern dark theme with gradient headers

## 🛠️ Tech Stack
- **Language:** Java
- **GUI:** Java Swing, AWT
- **OOP Concepts:** Inheritance, Polymorphism, Abstraction, Encapsulation
- **File I/O:** Data persistence using `.txt` files

## 🚀 How to Run
1. Clone: `git clone https://github.com/ishanayyab71-netizen/Vehicle-Rentals-System-Java-OOP.git`
2. Compile: `javac VehicleRentalSystem.java`
3. Run: `java VehicleRentalSystem`
4. Login: `Isha` / `Isha123`
